# Access Control Plugin for Baserow

A fine-grained access control plugin for Baserow that provides comprehensive permission management at multiple levels.

## Features

- **Workspace Level**: Plugin permissions and structure permissions (create/delete databases)
- **Database Level**: Member collaboration settings and table access control
- **Table Level**: Read-only/editable permissions and field structure permissions
- **Field Level**: Hidden/read-only/editable permissions per user
- **Row Level**: Invisible/read-only/editable permissions per user
- **Conditional Rules**: Dynamic row permissions based on field values

## Installation

1. Copy the plugin to your Baserow plugins directory
2. Add the plugin to your Baserow configuration
3. Restart Baserow
